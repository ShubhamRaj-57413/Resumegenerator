# Java Resume Generator 📄

A simple, console-based Java application that takes user input and generates a neatly formatted resume in the terminal.

## ✨ Features
* **User Input:** Collects name, contact details, education, and experience.
* **Professional Layout:** Automatically aligns sections for a clean look using custom borders.
* **Instant Preview:** View your formatted resume directly in your command line.

## 🚀 How to Run
1. Make sure you have **Java JDK** installed on your computer.
2. Download or copy the `ResumeGenerator.java` file.
3. Open your terminal and compile the code:
   ```bash
   javac ResumeGenerator.java
